# 📋 K-OO 커리어 개발 플랫폼

> **한 줄 소개:** K-OO 훈련생들의 체계적인 역량 진단과 커리어 성장을 지원하는 통합 학습/취업 지원 플랫폼
>
> **Live Site:** [https://ubion-project-git-main-wiky86s-projects.vercel.app/]

---

## 💡 프로젝트 개요 (Project Overview)

* **개발 기간:** 2025.10 ~ 2025.10
* **개발 인원:** 개인 프로젝트
* **기획 의도:**
  단순한 강의 수강을 넘어, 훈련생이 자신의 역량을 스스로 진단하고(커리어진단), 프로젝트 경험(미니프로젝트)과 스킬셋(커리어스킬)을 체계적으로 관리하여, 최종적으로 경쟁력 있는 포트폴리오를 완성할 수 있도록 돕는 **'올인원 커리어 플랫폼'** 구축

---

## 🛠️ 기술 스택 (Tech Stack)

* **Framework:** Next.js, React
* **Styling:** Tailwind CSS
* **Component:** Responsive Navigation Bar, Card UI
* **Deployment:** Vercel

## 📌 주요 기능 (Key Features)

상단 내비게이션 바의 7가지 핵심 메뉴를 중심으로 기능을 구현했습니다.

### 1. 커리어 진단 (Career Diagnosis)
* **역량 평가:** 현재 기술 수준과 직무 적합도를 진단하는 테스트 기능 제공
* **진단 리포트:** 평가 결과를 시각화된 그래프로 제공하여 강점과 보완점 분석

### 2. 미니 프로젝트 (Mini Projects)
* **프로젝트 관리:** 추천 주제를 통한 미니 프로젝트 체험
* **결과물 아카이빙:** 프로젝트 산출물을 업로드하고 관리하는 저장소 기능

### 3. 교육 과정 (Curriculum)
* **과정 목록:** 현재 수강 가능한 교육 과정 정보 및 커리큘럼 조회
* **학습 현황:** 개인별 진도율 및 출결 현황 확인

### 4. 커리어 스킬 (Career Skills)
* **스킬 트리:** 직무별 필수 기술 스택을 로드맵 형태로 시각화
* **학습 가이드:** 각 스킬 습득을 위한 추천 학습 자료 및 가이드 제공

### 5. 포트폴리오 (Portfolio)
* **포트폴리오 빌더:** 누적된 프로젝트와 스킬 데이터를 바탕으로 이력서 및 포트폴리오 자동 생성 지원
* **피드백:** 포트폴리오 첨삭 피드백 기능

### 6. 지식 정보 (Knowledge Base)
* **Tech News:** 최신 IT 트렌드, 채용 정보, 기술 아티클 큐레이션 제공
* **자료실:** 학습에 필요한 참고 자료 및 서식 다운로드

### 7. 기관 소개 (About)
* **교육 철학:** K-OO 운영 기관의 비전 및 교육 목표 소개
* **강사진 소개:** 분야별 전문 멘토 및 강사진 프로필 안내

## 🚀 트러블 슈팅 & 배운 점 (Challenges & Solutions)

### 1. 프레임 설계
* **문제:** 메뉴의 구성과 구현 방식
* **해결:** 메뉴별 목표 명확화 및 유사 사례 참조

### 2. 공통 UI 컴포넌트 설계를 통한 생산성 향상
* **문제:** '교육 과정', '미니 프로젝트', '지식 정보' 등 7개의 메뉴가 각기 다른 데이터를 다루지만, 화면에 표시되는 형태(카드 리스트)는 매우 유사하여 코드 중복 발생
* **해결:** 각 메뉴별로 카드를 따로 만드는 대신, 공통 컴포넌트 설계
